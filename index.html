RPi-sysinfo

Quick system information script for the Raspberry Pi

Download this project as a .zip file Download this project as a tar.gz file

Ever wanted to view your Raspberry Pi's stats all at once?

RPi-sysinfo is a script that shows you at a glance:

  • System uptime
  • Processor information
    • Clock speeds (minimum, current and maximum)
    • CPU governer
  • CPU and CPU temperature
  • Network information
    • IP addresses
    • MAC addresses
    • External IP address
  • Memory (used, free and swap)
  • Disk usage

Usage

  1. Run the install script
  2. Call sysinfo
